HTML与CSS 基础知识总结

编写HTMl与CSS时需要养成的习惯：

1.先套用Normalize.css，增强网页的兼容性
2.注意规范化命名标签和清晰简明的注释，还有代码排版
3.规范每个元素的属性排列顺序
4.注意网站文件的存储结构

===

html的基本结构

===

###常用元素功能及常用属性一览
<h> </h> 标题，块元素。字体从大到小有h1~h5，可自行更改样式
<p> </p> 常用的段落元素，块元素。
<a> </a>

===

元素：内联元素 块元素和空元素

*块元素：生成元素框，填充父元素的内容区，前后自带分隔

*内联元素：生成元素框，不会打破文本结构

*空元素：没有内容和结束标记
